# Akıllı Restoran Öneri ve Harita Uygulaması

Kullanıcının anlık konumunu tespit ederek çevresindeki restoran, kafe ve yemek
mekanlarını interaktif bir harita üzerinde listeleyen; kişisel tercihlere göre
özelleştirilmiş mekan önerileri sunan bir web uygulaması.

> 🚧 Proje aktif geliştirme aşamasındadır.

## Özellikler

- 📍 Tarayıcı Geolocation API ile anlık konum tespiti
- 🗺️ Harita üzerinde dinamik işaretçiler (Leaflet + OpenStreetMap)
- 🔎 Konuma göre gerçek zamanlı mekan verisi (Overpass API)
- ⚙️ Mutfak türü, mesafe ve fiyat aralığına göre tercih profili
- ⭐ Kişiselleştirilmiş öneri ve puanlama algoritması
- 💾 Tercihlerin ve favorilerin yerel olarak saklanması

## Teknoloji Yığını

**Frontend:** React, Vite, TypeScript, Tailwind CSS, shadcn/ui, Zustand
**Backend:** Python, FastAPI
**Harita & Veri:** Leaflet, OpenStreetMap, Overpass API
